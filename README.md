# Falcon-AI
**Falcon-AI** is a long-range aircraft image classifier inspired by the Azerbaijan plane crash. This AI system is designed to classify aircraft into three categories:

- **Civilian Plane**
- **Military Plane**
- **UAV (Unmanned Aerial Vehicle)**
